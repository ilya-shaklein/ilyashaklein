# Лендинг Ильи Шаклеина - маркетинг полного цикла

Второе позиционирование (широкий маркетинг, любые ниши/бюджеты). Отдельно от growthnottraffic.

## Перед деплоем - заполнить

**1. Контакты в `index.html`** (блок CONFIG вверху `<script>`):
```js
const TG_USER = "ilya_shaklein";   // уже стоит
const WA_PHONE = "77000000000";    // ПОМЕНЯТЬ на свой номер WhatsApp (без + и пробелов)
```

**2. ENV-переменные в Vercel** (Settings -> Environment Variables):
- `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather (для формы заявок)
- `TELEGRAM_CHAT_ID` - твой chat_id, куда падают заявки
- `ANTHROPIC_API_KEY` - для аудита (как на growthnottraffic)

### Как получить TELEGRAM_CHAT_ID
1. Создай бота у @BotFather, скопируй токен.
2. Напиши своему боту любое сообщение.
3. Открой `https://api.telegram.org/bot<ТОКЕН>/getUpdates` - там будет `"chat":{"id": ...}`. Это и есть chat_id.

## Деплой
Как обычно: залить папку на GitHub (Upload files -> Commit), Vercel подхватит автодеплоем.
Или новый проект в Vercel из этого репо.

## Структура
- `index.html` - лендинг (RU/EN, статика)
- `api/lead.js` - форма -> Telegram-бот
- `api/audit.js` - экспресс-аудит лендинга через Claude
- `package.json` - ESM для serverless

## Что заполнить в кейсах (плейсхолдеры `[__]`)
В блоке кейсов часть метрик стоит заглушками - подставь реальные:
- **EdTech:** `+[__]%` рост заявок
- **Mobile app:** `[__]%` payback, `[__]` CAC/LTV
- **E-commerce (Fungiline):** ROAS, CPO, `[__]%` рост продаж + в тексте `[вставь 1-2 цифры]`
Искать в `index.html` по `[__]`. Если цифр нет - лучше убрать метрику, чем ставить липовую.
